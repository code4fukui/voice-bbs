# voice-bbs

Denoで構築された、音声ベースのミニマルな掲示板システムです。

このプロジェクトは、ユーザーが短い音声メッセージを投稿・再生できる、シンプルで自己完結型のWebアプリケーションです。すべてのデータはローカルの `data.json` ファイルに保存されます。

## 機能

- **音声メッセージ**: ブラウザから直接音声メッセージを録音して投稿できます。
- **再生**: 他のユーザーが投稿したメッセージを再生できます。
- **シンプルなUI**: Sakura.cssでスタイリングされた、クリーンなシングルページインターフェースです。
- **軽量なバックエンド**: 最小限のDenoサーバーがAPIリクエストを処理します。
- **ローカルでのデータ永続化**: メッセージはローカルの `data.json` ファイルに保存されます。

## 仕組み

本アプリケーションは、Denoサーバーと静的なフロントエンドで構成されています。
1. フロントエンドは [`<input-voice>`](https://github.com/code4fukui/input-voice) Webコンポーネントを使用し、音声をWAVファイルとしてキャプチャします。
2. ユーザーが投稿を送信すると、音声データはBase64エンコードされ、`/api/add` エンドポイントに送信されます。
3. Denoサーバーは、新しい投稿（名前、音声データ、タイムスタンプ）を `data.json` に追記します。
4. ページ読み込み時に、フロントエンドが `/api/list` からすべての投稿を取得し、掲示板を表示します。

## 前提条件

- [Git](https://git-scm.com/)
- [Deno](https://deno.land/)

## はじめに

1. **リポジトリのクローン:**
    ```bash
    git clone https://github.com/code4fukui/voice-bbs.git
    ```

2. **プロジェクトディレクトリへの移動:**
    ```bash
    cd voice-bbs
    ```

3. **サーバーの起動:**
    ```bash
    deno run -A --unstable voicebbs.js
    ```

4. **アプリケーションを開く:**
    Webブラウザで [http://localhost:8008](http://localhost:8008) にアクセスします。

## APIエンドポイント

サーバーは以下の2つのシンプルなAPIエンドポイントを提供しています。

- `GET /api/list`
    - すべての投稿オブジェクトのJSON配列を返します。
- `POST /api/add`
    - 新しい投稿を追加するためのJSONペイロードを受け付けます。
    - ペイロードの形式: `{ "name": "string", "wav": "base64-string", "date": "string" }`

データは `data.json` に保存（永続化）されます。このファイルはバージョン管理から除外されています。
